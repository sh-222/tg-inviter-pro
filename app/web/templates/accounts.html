Add New Account

Uploading a pre-authorized .session file is the fastest way to add an account without requiring a login code.

Instructions

1. API Credentials: You can get your api_id and api_hash from my.telegram.org.

2. Proxies: Highly recommended to use 1 dedicated IPv4 proxy per account to avoid chain-bans. Format supports SOCKS5 and HTTP.

3. Sessions: We currently support uploading standard Pyrogram/Kurigram `.session` files. Make sure the phone number matches the session.
